Java library for accessing Sdb files. {@link org.lcogt.sdb.SdbParserApp} shows example usage of the library classes. To simply parse and Sdb file:- 
/* Open Sdb file */
SdbFile sdbfile = new SdbFile(datafile);

/* Parse the sdb */
while ((SdbDatum datum = sdbfile.readNextDatum()) != null)
{
   System.out.print(datum.toString());
}

sdbfile.closeFile();
